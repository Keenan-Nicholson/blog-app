# Local Development

## Run the Docker container to host the database

```bash
docker compose up -d
```

## Start the Frontend

1. Install dependencies:

```bash
npm install
```

2. Run the dev server

```bash
npm run dev
```

## Start the Backend

1. Install the dependencies:

```bash
npm install
```

2. Run the Express server

```bash
node server.ts
```
